Webserv

A lightweight HTTP server built in C++ as a group project at 42 Berlin.

About the project

Webserv is a modular, educational HTTP server built in C++ for the 42 Berlin curriculum. It demonstrates core web server concepts such as socket programming, non-blocking I/O, event-driven architecture, and CGI support. Webserv serves static files, executes CGI scripts, handles file uploads, and provides custom error pages. The server is configured using flexible location blocks, allowing for per-path settings and easy extensibility. This demo site showcases static content, team information and basic navigation, illustrating how Webserv can power modern web pages and assets.

Meet our team

We met at 42 Berlin at the very start of our journey, going through the piscine together and building a friendship that grew alongside our skills. Over time, we each followed our own pace through the curriculum, shaped by different priorities, life situations, and learning styles. Webserv was finally the opportunity to bring us all together and share our different strengths to create something meaningful that marks an important step in our path toward becoming developers.
