Webserv

Webserv is a lightweight yet fully featured C++ web server. It implements key HTTP functionalities, supports multiple virtual hosts, and is designed to teach you how web servers handle requests and responses. You’ll learn about socket programming, non‑blocking I/O, and event loops while building a production‑ready server. Webserv is a lightweight, educational HTTP server from 42 Berlin core corriculum which is written in C++. It supports static file serving, CGI scripts, file uploads, and custom error pages. This demo site shows static content and basic navigation.
